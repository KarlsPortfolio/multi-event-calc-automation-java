Multi-Event Calculator Testbed (Java)  Project Overview This repository serves as a Quality Assurance Laboratory. 

The primary objective is to take a functional Decathlon & Heptathlon Calculator (built with Java Spring and JavaScript) and wrap it in a comprehensive, 
full-stack automation ecosystem.Unlike a standard project, this "Testbed" is designed to compare different automation methodologies (like Selenium vs. Playwright) 
and demonstrate mastery of the Test Automation Pyramid.

The Lab Roadmap I am currently implementing the following testing layers to achieve 100% confidence in the software's stability.

    Unit Testing: JUnit 5 for core scoring logic and coefficient calculations.[ ] 
    API Testing: REST Assured for verifying Spring Boot endpoints and JSON payloads.[ ] 
    UI Automation (Web): A head-to-head comparison between Selenium WebDriver and Playwright.[ ] 
    UI Automation (Desktop): Exploring JavaFX testing via TestFX.[ ] 
    CI/CD Integration: Automated test execution via GitHub Actions. [ ]
